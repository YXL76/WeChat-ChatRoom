# WeChat-ChatRoom

## 介绍

微信聊天室小程序，花了`3`天左右的时间开发的。

前端配色来自`TIM`，不够完善，比如键盘升起时的界面、消息框的位置。

后端用到了`python`的[simple-websocket-server](https://github.com/dpallot/simple-websocket-server)

## 安装

1. 启动`server`文件夹中的`python`脚本

   ```
   python3 server.py
   ```

2. 将`client`文件夹导入微信开发者工具

3. 修改`AppID`

4. 修改`chat.js`中的`url`
